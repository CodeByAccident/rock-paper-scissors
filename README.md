# 🎮 Rock Paper Scissors (Python CLI Game)

A simple and fun command-line implementation of the classic **Rock, Paper, Scissors** game built using Python.

---

## 🚀 Features

* Play against the computer 🤖
* Randomized computer choices
* Input validation (no crashes on wrong input)
* Replay option to keep playing
* Clean and interactive terminal experience

---

## 🛠️ Tech Stack

* Python (Standard Library)
* `random` module

---

## ▶️ How to Run

1. Make sure Python is installed
2. Clone the repository:

   ```bash
   git clone https://github.com/your-username/rock-paper-scissors.git
   ```
3. Navigate to the folder:

   ```bash
   cd rock-paper-scissors
   ```
4. Run the script:

   ```bash
   python main.py
   ```

---
